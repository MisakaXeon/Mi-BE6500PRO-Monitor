# 更新日志

[English](CHANGELOG_EN.md) | 简体中文

本文件记录 Mi BE6500 Pro Monitor 的重要功能、修复和兼容性变更。

## [1.1.0] - 2026-07-16

### 修复

- 修复程序长时间运行后 Web 面板和 `/metrics.json` 可能需要数分钟才能响应的问题。
- 为 HTTP 监听器加入周期性唤醒机制，避免小米路由器内核环境中遗漏监听事件后长期阻塞。
- 增加读取请求头、读取、写入和空闲连接超时，避免异常连接长期占用服务资源。
- 将内嵌面板和 ECharts 文件在服务启动时读入内存，避免每次请求重复读取资源。
- 加固服务 PID、锁文件和后台进程识别逻辑，降低重复启动、错误停止和失效锁残留的概率。

### 新增

- 新增 `rmmon check-update`、`rmmon update` 以及管理菜单中的在线检查和在线更新入口。
- 新增版本文件和 `router-monitor -version`，管理菜单会显示当前版本。
- 新增 GitHub Raw 与 jsDelivr 更新源自动切换。
- 新增独立日志文件及 `rmmon log` 查看入口。

### 安全与可靠性

- 安装包和更新包使用固定文件清单进行 SHA-256 完整性校验。
- 更新采用事务式替换：先备份，再安装和启动；启动或健康检查失败会自动恢复旧版本。
- 更新时保留监听端口、采集间隔、更新源和开机自启设置。
- 默认仅允许 HTTPS 更新源；HTTP 只用于显式开启的不安全测试模式。
- 配置文件按字段白名单解析，不再作为 Shell 脚本直接执行。
- 安装目录限制在 `/data/other_vol/` 下，并阻止路径越界和符号链接绕过。

### 测试与构建

- GitHub Actions 自动运行 Go 单元测试、集成测试、BusyBox `ash` 兼容性测试和发布文件校验。
- 自动构建 Linux ARM64 二进制并更新 `checksums.txt`。
- 已在小米 BE6500 Pro 上验证安装、重装、在线检查、端口冲突回滚和服务健康状态。

更早版本尚未维护独立更新日志，可通过仓库提交记录查看历史。
